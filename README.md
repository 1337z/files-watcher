# files-watcher

## :star: Features

![Preview](https://i.imgur.com/5Tvj6S6.gif)

Run `files-watcher` to get notified on file changes in your current directory.  
Useful for debugging, finding maleware and performance managing.

## :package: Install

```command
> npm i -g files-watcher
```

## :clipboard: Usage

```command
> files-watcher
```
